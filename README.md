## 使用GitHub

* [申請GitHub帳號](https://git-scm.com/book/zh-tw/v2/GitHub-建立帳戶及設定)
* 至此連結 [https://github.com/tool-jx3/jx3-mj](https://github.com/tool-jx3/jx3-mj)
* [Fork專案](https://git-scm.com/book/zh-tw/v2/GitHub-參與一個專案)，並於自己Fork後的專案修改內容
* 發Pull Request至原專案
